Septembre 2020

Page "Nouveautés"

La nouvelle page "Nouveauté" répertorie toutes les modifications notables sur l'outil ADE-Scheduler. N'hésitez pas à jeter un coup d'oeil de temps en temps pour voir les nouvelles fonctionnaliés !

Palette de couleurs accessibles dans le calendrier

Nous devons admettre que modifier la couleur des évènements du calendrier depuis la page "Compte" n'était pas très intuitif... C'est pourquoi, il est désormais possible de choisir vos propres couleurs pour les cours/TPs/autres ! Et ce, même sans devoir posséder un compte. De plus, nous avons aussi changé le thème couleur initial, afin d'accentuer la différence entre deux cours.

Notez tout de même que si vous utilisez un navigateur particulier, il est possible que la palette permettant de modifier les couleurs à votre guise ne soit pas disponible... Cela est du au navigateur.

Tous les horaires accessibles depuis le calendrier

Vous pouvez désormais consulter et modifier tous vos différents horaires sauvegardés sans devoir passer par le menu "Compte". Si vous possédez plusieurs calendrier, il suffit pour cela de cliquer sur le nom de l'horaire affiché actuellement pour voir défiler le nom de tous vos horaires; il suffit ensuite de sélectionner celui de votre choix.
